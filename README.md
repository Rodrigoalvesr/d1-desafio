## Scripts

Execute no diretório principal do projeto:

### `yarn start`

Executa o aplicativo no modo de desenvolvimento.

### `yarn test`

Inicia os teste no modo de observação interativo.

### `yarn build`

Compila o aplicativo para produção na pasta `/build`.
